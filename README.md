# Galax Client

## Getting Started

### Prerequisites

- [Git](https://git-scm.com/)
- [Node.js and npm](https://nodejs.org/en/) Node ^6.2.0, npm ^3.8.9
- [React](https://github.com/facebook/react)
- [Webpack](https://webpack.github.io/)
- [ES6](http://es6.ruanyifeng.com/#docs/intro)
- [BookList](https://github.com/ericdouglas/ES6-Learning)
- [Yarn](https://yarnpkg.com/)

### Setup

1. Run `make install` to install dependencies.


### Develop

- Run `make run` to start project


### Build

- Run `make build-ci` to build ci package
- Run `make build-live` to build prod package


### Deploy Before Test

- Run `make pre-deploy`
